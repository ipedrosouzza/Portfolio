Meus Projetos

Landing Page "FocusFlow"

Este projeto apresenta o design e desenvolvimento de uma landing page moderna para o aplicativo fictício FocusFlow. A proposta foca em uma experiência de usuário clara e intuitiva, com uma estrutura visual baseada em hierarquia tipográfica, ícones circulares para etapas, cores contrastantes e espaçamento confortável. A página segue o conceito de "design orientado à ação", conduzindo o visitante em três passos simples para entender o produto. Foi aplicada responsividade completa, garantindo consistência em desktop e dispositivos móveis.
